## 4. 求平均时间

```javascript
const arr = ["8:01", "9:30", "11:50"];
let res = 0;
arr.forEach(el => {
  res += new Date('2000/01/01 ' + el).getTime();
})
const average = new Date(res / arr.length);
console.log(`${average.getHours()}:${average.getMinutes()}`);
```
